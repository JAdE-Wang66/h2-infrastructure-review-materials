# Supporting Materials for Peer Review

This repository contains the supporting materials underlying the manuscript under review.

## Folder structure

- `code/`  
  Scripts used for data processing, analysis, and generation of results.

- `data/`  
  Processed data used in the analysis, together with related supporting files where applicable.

- `output/`  
  Output files corresponding to the main results reported in the manuscript, including selected tables, figures, or model outputs.

## Notes

These materials are provided to support editorial assessment and peer review of the manuscript.

Some raw source data may be subject to third-party restrictions and therefore may not be redistributed directly. Where applicable, processed data and supporting materials are provided instead.
